## 📌 Problem Statement

Industrial machines fail unexpectedly, causing:
- 🔴 Unplanned downtime costing thousands per hour
- 🔴 Emergency repair costs
- 🔴 Production line shutdowns
- 🔴 Safety hazards for workers

GUARDIAN predicts failures **before they happen** using real-time
sensor data, allowing factories to schedule preventive maintenance
and save costs.

## ⚙️ Sensor Input Panel Explained

The sidebar contains 6 real industrial sensor inputs:

| Sensor | Range | What it measures |
|---|---|---|
| 🏭 Machine Grade | L / M / H | Quality grade of product being manufactured |
| 🌡️ Air Temperature | 295–305 K | Ambient room temperature around the machine |
| 🌡️ Process Temperature | 305–315 K | Heat generated by the machine while working |
| ⚡ Rotational Speed | 1168–2886 RPM | How fast the machine spindle rotates |
| ⚡ Torque | 3.8–76.6 Nm | Force applied by the machine |
| 🔩 Tool Wear | 0–253 min | How long the cutting tool has been in use |

### How sensors relate to failures:
- **Air vs Process Temp difference > 11K** → Heat Dissipation Failure risk
- **High Torque + Low RPM** → Power Failure risk
- **High Torque + High Tool Wear** → Overstrain Failure risk
- **Tool Wear > 200 min** → Tool Wear Failure risk

## 📊 Analysis Dashboard Explained

### 1. 🎯 Risk Score Card
The main prediction output from the XGBoost model.
- 🟢 **0–30%** → Machine is healthy, continue normal operation
- 🟡 **30–70%** → Warning, schedule maintenance soon
- 🔴 **70–100%** → Critical, stop machine immediately

### 2. 🔩 Tool Wear Card
Shows what percentage of the tool's total lifespan has been consumed.
- 🟢 Below 60% → Tool is fine
- 🟡 60–80% → Plan replacement soon
- 🔴 Above 80% → Replace immediately

### 3. 🌡️ Thermal Delta Card
The temperature difference between process and air temperature.
- Normal range is below 11K
- Above 11K → Heat dissipation failure risk

### 4. 📈 Gauge Chart
A visual speedometer showing the overall failure risk score with
color-coded zones from safe (green) to critical (red).

### 5. 🕸️ Radar Chart
Shows all 5 sensors simultaneously in one view.
- The orange shape = current sensor readings
- The dotted line = normal operating range
- Larger shape = more stressed machine

### 6. 🔬 Failure Type Breakdown
Identifies which specific type of failure is most likely:

| Failure Type | Cause |
|---|---|
| Tool Wear Failure | Tool used beyond safe operating limit |
| Heat Dissipation Failure | Machine overheating, poor cooling |
| Power Failure | Abnormal power consumption pattern |
| Overstrain Failure | Excessive torque on a worn tool |
| Random Failure | Unpredictable mechanical failure (always ~3%) |

### 7. 💰 Maintenance Cost Estimator
Compares two scenarios side by side:

| Ignored Failure Cost | Preventive Maintenance Cost |
|---|---|
| Unplanned downtime | Scheduled service |
| Emergency repair | Parts replacement |
| Production loss | — |

Shows exactly how much money is saved by acting early.

### 8. 🔍 SHAP AI Explainability
Explains **why** the model made its prediction using SHAP values:
- **Blue bars** → Feature pushing prediction towards healthy
- **Red bars** → Feature pushing prediction towards failure
- **Longer bar** → More influence on the final prediction

This makes the AI transparent and trustworthy — critical for
industrial deployment.

### 9. 📋 Recommended Actions
Clear action items based on risk level:
- 🚨 **Above 70%** → Stop machine, contact maintenance team immediately
- ⚠️ **40–70%** → Schedule maintenance within 24 hours
- ✅ **Below 40%** → Continue normal operation, all systems nominal
